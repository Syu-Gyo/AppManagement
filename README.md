# AppManagement（デモ版）

全社のソフトウェアライセンスを「誰が・どのアプリを・いくらで」使っているかを一元管理するための Web アプリです。
React + Vite + TypeScript 製。`2026全国デザイン設計部員.xlsx` の内容をシードデータとして取り込んでいます。

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 が自動で開きます。

## 画面構成

| 画面 | 内容 |
| --- | --- |
| 📊 ダッシュボード | メンバー数 / ソフト数 / ライセンス付与数 / 月額コスト（概算）、ソフト別・部署別の集計 |
| 👥 メンバー | 社員一覧。名前・部署・ソフトで検索／絞り込み。行クリックで詳細・編集 |
| 📦 ソフトウェア | アプリごとの利用者数とコストをカード表示 |
| 🤖 AIツール | AI配布プラン（Midjourney/ChatGPT/KreaAI/Genspark/Tripo/Adobe Express/Google AI）の契約本数・配布先・金額・契約/更新日・アカウント管理者を管理。本数＜配布人数の超過を警告 |
| 🔑 API | APIキーの利用量・月予算・予算消化率・状態（有効/無効）を管理。予算逼迫アラート |
| 🔲 ライセンス表 | メンバー × ソフトのマトリクス。セルをクリックで付与／解除 |
| 📅 契約スケジュール | ソフト契約の更新時期・本数・コスト管理。更新アラート、年間ガントチャート、契約本数と実利用者数の突合 |

## デモの仕様

- データはブラウザの localStorage に保存されます（追加・編集・削除が反映されます）。
- サイドバーの「デモデータをリセット」で初期状態（Excel取り込み時点）に戻せます。
- ソフトの**月額金額はデモ用の仮置き**です（`src/data/software.ts` で調整可能）。
- バックエンドは未接続のフロントエンドのみの構成です。

## データの差し替え

- 社員データ: `src/data/seed.ts`（`2026全国デザイン設計部員.xlsx` から自動生成）
- ソフトのカテゴリ・金額・ベンダー: `src/data/software.ts`
- 契約スケジュール: `src/data/contracts.ts`（`2026ソフト-AI契約スケジュール.xlsx` のソフト名・本数を基にした編集可能なデモデータ。契約期間・更新日・単価はデモ用の仮置き）
- AIツール: `src/data/aiplans.ts`（`各AI配布プラン.xlsx` から自動生成。契約プラン・本数・配布先・契約/更新日・管理者。メール/パスワードは取り込まず、金額は$1=155円換算の概算）
- API: `src/data/apikeys.ts`（APIキー・利用量のデモデータ。利用額/予算は概算、キーはマスク表示のみ）

> 契約スケジュールの基準日は `src/components/Contracts.tsx` の `TODAY`（2026-06-16）で固定しています。
